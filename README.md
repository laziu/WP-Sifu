# Sifu

### 디렉터리 구조
```
├── Blueprints
│   ├── Anims      <- Animation Blueprints (ABP_*)
│   ├── Core       <- GameInstance, core utilities
│   ├── StateTree  <- StateTree assets (ST_*, STCond_*, STTask_*, STEval_*)
│   └── Widgets    <- UI (WBP_*)
├── Characters  <- Animation Sequences, Meshes, etc. for each character
├── Effects     <- Particle Systems (P_*), Niagara Systems (NS_*), etc.
├── Input
├── Maps
├── Materials   <- Material (M_*), Material Instances (MI_*)
├── Media       <- Sound, Video, etc.
├── Meshes      <- Static Meshes (SM_*) without skeletal
└── Textures    <- Textures (T_*) without material
```

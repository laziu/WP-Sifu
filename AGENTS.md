## Project Overview

Sifu 게임을 UE5로 모작하는 프로젝트입니다.

## Instruction

- 직접적인 구현 및 수정을 지시하지 않을 경우 파일을 직접 수정하지 마세요.
- 변경해야 할 코드를 식별할 때마다, 검토할 수 있도록 해당 스니펫을 응답에 포함하세요.
- 디자인에 대한 가정을 해야 할 때마다, 잠시 멈추고 저에게 질문하세요.
- 많은 양의 코드를 변경해야 할 경우, 별도의 markdown 파일로 계획을 작성하여 검토할 수 있도록 하세요. 해당 파일을 사용자가 수정하여 계획을 개선할 수 있도록 하세요.
- 기능을 구현할 땐 가급적 UE5 내장 기능만 사용하세요. 외부 라이브러리를 사용해야 할 경우, 반드시 사전에 저에게 알려주세요.
- Blueprints, Input, Animation 등 사용자가 수동으로 수정해야 하는 부분이 있다면, 해당 부분을 명확히 설명하세요.
- SOLID 규칙을 준수하여 ActorComponent 등을 활용하세요.

## Setup

- Windows 11, Unreal Engine 5.7 버전 사용
